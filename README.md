<h2>HR Analytics: Predicting Employee Satisfaction and Attrition</h2>

<p>
Система HR-аналитики для прогнозирования удовлетворённости сотрудников и оценки риска их увольнения.
</p>

<h3>Задача</h3>

<p>
Снизить текучесть кадров и повысить вовлечённость сотрудников за счёт раннего выявления факторов неудовлетворённости и прогнозирования вероятности увольнения.
</p>

<h3>Решение</h3>

<p>
Разработал две модели машинного обучения: первую — для прогнозирования уровня удовлетворённости сотрудников, вторую — для оценки риска увольнения.
</p>

<p>
Провёл исследовательский анализ данных, обработал пропуски и неявные дубликаты, настроил пайплайны предобработки и выполнил подбор гиперпараметров.
</p>

<p>
Для повышения качества прогнозирования увольнений добавил в модель новый признак — предсказанный уровень удовлетворённости сотрудника.
</p>

<p>
Интерпретировал результаты с помощью SHAP и сформировал портрет сотрудников с высоким риском увольнения.
</p>

<h3>Результат</h3>

<ul>
  <li>Достиг значения <strong>SMAPE = 13.75%</strong> при прогнозировании удовлетворённости сотрудников.</li>
  <li>Построил модель прогнозирования увольнений с качеством <strong>ROC-AUC = 0.932</strong>.</li>
  <li>Выявил ключевые факторы, влияющие на увольнение: уровень заработной платы, удовлетворённость работой, отсутствие повышения и стаж работы.</li>
  <li>Определил группы сотрудников с повышенным риском оттока.</li>
  <li>Сформировал рекомендации по снижению текучести кадров на основе анализа данных.</li>
  <li>Разработал воспроизводимый ML-процесс для поддержки HR-решений.</li>
</ul>
